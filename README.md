# 📊 Signal & Flow — by SW

A weekly, data-driven analysis of financial markets, macro trends, and capital flows.

---

## 🧠 What This Is
Signal & Flow is a structured market brief focused on two core ideas:

- 📈 **Signal** — what the data is saying  
- 🌊 **Flow** — where capital is moving  

Each report combines data visualization with concise analysis to identify high-conviction trends in markets.

---

## 🎯 Focus Areas
- 📊 Market performance (S&P 500, NASDAQ)
- 📉 Interest rates and macro trends
- 🌊 Sector rotation and capital flows
- 🚀 High-conviction investment themes
